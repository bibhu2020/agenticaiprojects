## ğŸ§© Types of Machine Learning

Machine Learning can be categorized into three main types:

### 1. **Supervised Learning**
The model is trained on labeled data â€” meaning the correct answer is known in advance.

Examples:
- Predicting house prices (`regression`)
- Classifying emails as spam or not spam (`classification`)

Popular algorithms: **Linear Regression, Decision Trees, Random Forest, Support Vector Machines (SVM)**

### 2. **Unsupervised Learning**
The model works on **unlabeled data**, discovering patterns or groupings on its own.

Examples:
- Customer segmentation
- Market basket analysis

Popular algorithms: **K-Means Clustering, Hierarchical Clustering, PCA**

### 3. **Reinforcement Learning**
Here, an agent learns by interacting with its environment to achieve a goal. It receives **rewards** or **penalties** based on actions.

Examples:
- Game-playing AI (like AlphaGo)
- Robotics and autonomous driving

Popular algorithms: **Q-Learning, Deep Q-Network (DQN)**

## ğŸ’» Example: Predicting House Prices

Hereâ€™s a simple Python example using **scikit-learn**:

```python
from sklearn.linear_model import LinearRegression
import numpy as np

# Training Data (square feet vs price)
X = np.array([[1000], [1500], [2000], [2500], [3000]])
y = np.array([150000, 200000, 250000, 300000, 350000])

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Predict price for a 2200 sq ft house
predicted_price = model.predict([[2200]])
print(f"Predicted Price: ${predicted_price[0]:,.2f}")
```

Output:
```
Predicted Price: $275,000.00
```
